# Password Generator

A responsive password generator built using HTML, CSS, and JavaScript that creates secure passwords with customizable options.

## Features
- Generate strong random passwords
- Select uppercase letters
- Select lowercase letters
- Include numbers and symbols
- Adjustable password length
- Password strength indicator
- Copy password to clipboard

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Select the password length.
2. Choose the options (uppercase, lowercase, numbers, symbols).
3. Click **Generate Password**.
4. Copy the generated password.
